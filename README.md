## 🌐 Como Funciona a Tradução

O sistema de tradução foi implementado com **JavaScript puro** sem dependências externas:

1. **Estrutura de Dados**: Objeto com 170+ strings em 3 idiomas (en, pt, el)
2. **Atributo data-translate**: Cada elemento traduzível tem um atributo único
3. **Troca Dinâmica**: JavaScript atualiza todos os elementos instantaneamente
4. **LocalStorage**: Salva a preferência do usuário automaticamente
5. **Inicialização Inteligente**: Carrega o último idioma usado ao abrir o site
6. **Performance**: Troca de idioma em menos de 50ms

### Elementos Traduzidos:
- ✅ Menu de navegação
- ✅ Seção Hero (títulos, badges, botões)
